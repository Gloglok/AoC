# Advent of Code 2025

Solutions for Advent of Code 2025 in Elixir.

## Building and Running

This project is configured as an escript, allowing you to build and run it as an executable.

### Build the executable

```bash
mix escript.build
```

### Run the executable

```bash
./aoc2025
```

## Running Tests

Run all tests:

```bash
mix test
```

Run tests for a specific day:

```bash
mix test test/day1_test.exs
```

Run a specific test at a line number:

```bash
mix test test/day1_test.exs:line_number
```

## Project Structure

- `lib/day1.ex` - Day 1 solution
